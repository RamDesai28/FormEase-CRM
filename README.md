
# FormEase CRM

FormEase CRM is a modern, full-stack Customer Relationship Management (CRM) application built using the MERN stack. This project features a sleek dark theme, user-friendly forms, and a responsive admin panel for managing and viewing user data. The UI includes a sidebar for navigation and a form to input and display user information in a table format.

## Features

  - Responsive, dark-themed admin panel UI
  - Sidebar for easy navigation
  - Input form for collecting and saving user data
  - User data displayed in a table format
  - Full CRUD (Create, Read, Update, Delete) operations
  - Data stored using MongoDB
  - Backend API built using Node.js and Express
  - Frontend built with React.js
  - Designed with clean and modern styling, featuring relative color combinations
  
## Technologies Used

  - **Frontend**: React.js, CSS (custom styling)
  - **Backend**: Node.js, Express.js
  - **Database**: MongoDB (Mongoose)
  - **Other**: Axios (for API requests)
